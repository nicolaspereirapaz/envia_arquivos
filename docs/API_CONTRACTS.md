# Contratos da API

## POST /clients
Cria cliente

## POST /orders/simulate
Simula valor do pedido

## POST /orders
Cria pedido

## POST /uploads/photos
Envia fotos

## POST /uploads/documents
Envia documentos

## GET /orders/:id
Consulta pedido

## GET /admin/orders
Lista pedidos

## PATCH /admin/orders/:id/status
Atualiza status do pedido

## POST /whatsapp/order-summary/:id
Gera resumo do pedido e link de WhatsApp
