# Regras para qualquer IA

## Regras obrigatorias
- Nao criar logica fora dos modulos definidos
- Nao misturar frontend com regra de negocio
- Nao acessar banco direto no controller
- Nao duplicar DTOs
- Nao criar arquivos genericos gigantes como utils.ts
- Sempre usar TypeScript
- Sempre comentar decisoes importantes
- Sempre respeitar a estrutura oficial do monorepo
- Nao remover ou renomear modulos sem solicitacao explicita

## Ao gerar codigo
- Gerar primeiro interfaces e contratos
- Depois entidades
- Depois casos de uso
- Depois controllers
- Depois testes

## Ferramentas usadas no projeto
- ChatGPT
